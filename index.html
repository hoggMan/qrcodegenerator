QR Code Generator

This is a free resource for generating QR codes. Simply input a URL or text and click the button to generate a code. URLs will link to a site and text will display contents.

Legend

Text: Enter the text as is in the input field

URL: Enter the URL in the input field

Geo: Geo:[lat],[lng]

WiFi: WIFI:S:[SSID];T:[WPA|WEP|];P:[password];;
